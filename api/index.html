Bookri API Documentation

Welcome to the API documentation for the Bookri backend application. Below are the details for each endpoint.

POST api/auth/register

Description: Register a new user.

Mock URL: https://bookri.github.io/api/auth/register/post.json

Authentication: No

{
    "email": "user@example.com",
    "password": "securepassword",
    "name": "John Doe"
}

Response:

{
    "message": "User registered successfully",
    "userId": "12345"
}

POST api/auth/login

Description: Login and get a JWT token.

Mock URL: https://bookri.github.io/api/auth/login/post.json

Authentication: No

{
    "email": "user@example.com",
    "password": "securepassword"
}

Response:

{
    "token": "jwt-token-example",
    "expiresIn": 3600
}

GET api/kuri

Description: List all Kuri groups.

Mock URL: https://bookri.github.io/api/kuri/get.json

Authentication: Yes

Response:

[
    {
        "id": "kuri123",
        "name": "Monthly Savings Group",
        "members": 10,
        "amount": 1000,
        "rotationFrequency": "Monthly"
    },
    {
        "id": "kuri456",
        "name": "Weekly Savings",
        "members": 5,
        "amount": 500,
        "rotationFrequency": "Weekly"
    }
]

POST api/kuri/create

Description: Create a new Kuri group.

Mock URL: https://bookri.github.io/api/kuri/create/post.json

Authentication: Yes

{
    "name": "Holiday Savings",
    "amount": 2000,
    "rotationFrequency": "Monthly",
    "members": 8
}

Response:

{
    "message": "Kuri group created successfully",
    "kuriId": "kuri789"
}

POST api/kuri/join/{id}

Description: Join an existing Kuri group.

Mock URL: https://bookri.github.io/api/kuri/join/1/post.json

Authentication: Yes

Response:

{
    "message": "Successfully joined the Kuri group"
}

GET api/transactions/{kuriId}

Description: Get transaction history for a Kuri group.

POST api/payment

Description: Record a payment for a Kuri group.

Mock URL: https://bookri.github.io/api/payment/post.json

Authentication: Yes

    "amount": 1000,
    "paidBy": "John Doe"
}

{
